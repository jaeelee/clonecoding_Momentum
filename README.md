## memo
```
// Interval = 매번 일어나야 하는 무언가 (ex. 매 2초마다 무슨일이 일어나게 하고 싶을 떄 사용)
setInterval(sayHello, 5000); // 5초마다 sayHello 함수 실행
```
```
// timeout = 몇초 후에 일어나야 할 떄 사용 
setTimeout(sayHello, 5000); // 5초 후에 sayHello 함수 실행
```
